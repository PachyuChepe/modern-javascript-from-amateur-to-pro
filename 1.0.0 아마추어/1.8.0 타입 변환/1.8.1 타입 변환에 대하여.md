# 타입 변환
[1.5.1 데이터 타입에 대하여](https://bit.ly/3thgpoX)에서 자바스크립트의 모든 값은 데이터 타입을 갖는다고 하였다.  

코드 실행 중 데이터 타입이 변환되는 것을 **타입 변환**이라하며, 타입 변환에는 두 가지 형태가 있다.
1. **암묵적 타입 변환** (타입 강제 변환)  
   자바스크립트 엔진에 의해 암묵적으로 타입이 자동 변환된다.
2. **명시적 타입 변환** (타입 캐스팅)  
   개발자가 의도적으로 값의 타입을 변환한다.

> [1.8.2 암묵적 타입 변환과 명시적 타입 변환에 대하여](https://bit.ly/3nfuort)에서 살펴보자.
